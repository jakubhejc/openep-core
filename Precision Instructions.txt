2) Process and plot geometry

3) Load a single DxL file

[info, points, egms] = loadprecision_dxldata('/Users/Steven/Documents/Dropbox/NIHR_CL/People/JohnWhitaker/precision_Scrap/DxL_1.csv');

info - the header information at the top of the DxL file
points - the map data in the middle of the DxL file
egms - the electrogram data at the bottom of the DxL file

4) Load a folder of files

info = importprecision()
info = importprecision('/Users/Steven/Documents/Dropbox/NIHR_CL/People/JohnWhitaker/precision_Scrap')

info.dxldata is an array of structures containing info, points and egms data from loadprecision_dxldata

plot the geometries from a folder of files:
>> tr = processprecision_modelgroups(info.modelgroups);
>> plotprecision_modelgroups(tr, info.modelgroups);

DxL files are created by Precision when exporting 'Contact Map' data

Access the first point in the first file:
nFile = 1;
nPoint = 1;
p = info.dxldata(nFile).points(nPoint);

N.B. if nPoint >256; nPoint should be set to 1 and nFile set to 2 etc
